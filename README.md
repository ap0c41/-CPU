# CPU
用Basys3板和vivado实现简易的单多周期cpu
# 单周期CPU实验原理
单周期 CPU 要在一个周期内完成取址，译码，执行，访存，写回五个阶段。这五个阶段需要各个模块之间组合执行。首先要有一个指令寄存器存储需要用到的指令，还有一个 pc 地址计算模块来实现 pc 地址的更新。每次开始按照 pc 地址在指令寄存器取出对应的指令，将指令进行划分，前六位作为 opcode 来判断当前指令是什么指令（add，or，lw......），然后根据指令的判断给出相应的各个控制信号（PCWre，RegDst，aluop......），根据这些控制信号，运算器会选择运算的数据 src1 和 src2 和进行的运算类型，数据存储器会判断是否读或写，寄存器会判断是否写入。最后将 pc 等信号显示在数码管上即可。
